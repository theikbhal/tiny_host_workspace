# Getting Started

Welcome to the Tiiny Host API documentation. This API allows you to programmatically manage your projects on Tiiny Host.

## Generating Your API Key

To use the API, you need an API key.

1.  [Log in](https://tiiny.host/login) to your Tiiny Host account.
2.  Go to [Manage Account](https://tiiny.host/manage/account).
3.  Click on the **API Key** section.
4.  Click **Generate** to create a new API key.

> [!IMPORTANT]
> Keep your API key secure. Do not share it publicly.

## Prerequisites

Before you begin, ensure you have:

-   A Tiiny Host account.
-   A valid API key.
-   Basic knowledge of HTTP requests (e.g., using `curl`).

Check out the [pricing page](https://tiiny.host/pricing) for details on limits and features.
