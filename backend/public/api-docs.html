API Documentation

Welcome to the Tiny Host API. Use these endpoints to manage your static sites.

POST /upload

Uploads a ZIP file or a single HTML file to create a new site.

Request Body (multipart/form-data)

Field Type Description
folderName String Required. The name of the site (URL path).
overwrite Boolean Optional. Set to true to replace an existing site.
file File Required. The .zip or .html file to upload.

Example Response

{
  "message": "File uploaded and deployed successfully",
  "path": "/public/my-site"
}
DELETE /delete/:folderName

Deletes an existing site and all its contents.

Parameters

Parameter In Description
folderName Path Required. The name of the site to delete.

Example Response

{
  "message": "Site 'my-site' deleted successfully"
}
