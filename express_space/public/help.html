Getting Started

Welcome to Tiiny Host! Here's how to get your site online in minutes:

  • Create a ZIP file containing your index.html and assets
  • Go to the dashboard and choose a subdomain name
  • Upload your ZIP file
  • Click Launch and your site is live!

Common Questions

  • How do I update my site? Use the "Update Site" tab on the dashboard
  • Can I use a custom domain? Yes, available on Pro plans
  • What file types are supported? HTML, CSS, JavaScript, images, and other static assets
  • Is HTTPS included? Yes, all sites are automatically served over HTTPS

Troubleshooting

  • Site not loading? Make sure your ZIP contains an index.html file
  • Images not showing? Check that file paths are relative, not absolute
  • Upload failed? Ensure your ZIP is under the size limit (100MB for free, 1GB for Pro)
